# ✅ Практика 1 — CSS-препроцессоры (SASS)

📄 Основание: Методические указания к практическому занятию №1  

## 🎯 Цель работы  
Изучение CSS-препроцессоров (SASS/LESS) и применение:
- переменных  
- миксинов  
- вложенности  
- условных конструкций  

## 🔧 Что было реализовано

В рамках интернет-магазина реализована карточка товара с использованием SASS.

### Использованы возможности препроцессора:

✔ **Переменные (минимум 2)**  
- основные цвета проекта  
- размеры отступов  
- цвет кнопок  

✔ **Миксин (минимум 1)**  
Создан миксин для стилизации кнопки товара (фон, цвет текста, border-radius).

✔ **Вложенность селекторов**
```scss
.card {
  padding: 16px;

  &__title {
    font-size: 18px;
  }

  &__price {
    font-weight: bold;
  }

  &--accent {
    border: 2px solid $primary-color;
  }
}
```

✔ Условные конструкции (`@if`) для темы оформления  

✔ Компиляция SCSS → CSS через:
```bash
sass --watch styles.scss styles.css
```

## 📌 Результат  
Создана стилизованная карточка товара с применением возможностей CSS-препроцессора.

# ✅ Практика 2 — Сервер на Node.js + Express

## 🎯 Цель работы  
Создание REST API с использованием Express и реализация CRUD-операций.

## 🔧 Что реализовано

Создан сервер на **Node.js + Express**.

Подключены middleware:

✔ `express.json()` — обработка JSON  
✔ `express.urlencoded()` — обработка форм  
✔ собственное middleware для логирования запросов  

## 📦 Реализованы CRUD-операции для сущности Product

| Метод | Путь | Назначение |
|--------|--------|------------|
| GET | /products | Получение списка товаров |
| GET | /products/:id | Получение товара по ID |
| POST | /products | Создание товара |
| PATCH | /products/:id | Обновление товара |
| DELETE | /products/:id | Удаление товара |

Объект товара содержит:
- `id`
- `name`
- `price`

## 📌 Результат  
Реализован REST API с полным набором CRUD-операций.
